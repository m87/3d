## Supported batteries
All listed names are valid to be used in holder map.
- A / LR23 / R23
- AA
- AAA
- C
- D
- LR1 / LR01 / N / KN / AM5 / _4001 / _810 / MN9100 / E90 / _910A
- CR2 / DLCR2 / EL1CR2 / CR15H270
- CUSTOM (special name for custom battery size)
